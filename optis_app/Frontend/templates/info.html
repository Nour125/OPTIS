What does OPTIS do?

OPTIS is a standalone Python based web application which aims to minimize time requirements of busines processes.
It's customized to work on a single business process:

Assuming this is a real life business process, the user has event logs which follow different cases through their execution. The user will be interested in the so called "active" cases - the cases for which there are still decisions to be met. For example whether to choose activity "manufacture A" or "manufacture B", when the customer chose a custom order. Since this decision requires careful analysis of the available resources, the state of the other active cases, the time activities normally take etc, this decision is far more than trivial. Especially for bigger business processes, which can easily be modelled in the same way as this one.

OPTIS offers a solution to this problem by using reinforcement learning techniques to train a neural network, which recommends the optimal next activity for any active case in the event log. The network has been carefully trained using a simulation model based on the real business process (shown in the image above).

If you want to try our app but you don't have an event log which matches the process, we also offer the option to generate your own event logs using the previously mentioned simulation model. Just choose a duration period for the simulation and export your event log as .csv or .xes. Then you can upload it and start using our app.

After that you can preview the event log you uploaded and go to the case selection page if you're content with it. In the case selection page you can preview any of the active cases and choose the one which you want to optimize. After you select it you get your personalized recommendation from our trained network.

Have fun optimizing your process!
